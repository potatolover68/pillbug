A semi-automated editing tool for MediaWiki. [Getting started](https://github.com/potatolover68/pillbug/wiki/Getting-started)

## OAuth

When both `OAUTH_CONSUMER` and `OAUTH_SECRET` are set, Config shows **OAuth Login** instead of BotPassword fields.
